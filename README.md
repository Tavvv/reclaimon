# ReclaimOn — Marketing Website

Static marketing site for the ReclaimOn iOS app, live at [www.reclaimon.com](https://www.reclaimon.com).

Plain HTML/CSS/JS — no build step, no framework, no dependencies. Deploy = serve the repo root.

## Pages

- `index.html` — homepage
- `best-app-blocker-iphone.html` — SEO guide: choosing an app blocker for iPhone
- `reclaimon-vs-opal-freedom.html` — comparison vs Opal, Freedom & one sec
- `app-blocker-for-adhd.html` — ADHD & deep-work use case
- `privacy.html` — privacy policy

Plus `robots.txt` and `sitemap.xml` at the root.

Images live in `site-assets/`; the social card is `og-image.jpg` and the favicon is `favicon.png` (both at the repo root).

The previous React/Vite version of this site is preserved on the `archive/react-site` branch.
